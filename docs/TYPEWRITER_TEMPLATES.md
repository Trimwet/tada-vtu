## Usage Notes

The typewriter effect rotates through messages based on:
- **Time of Day**: Different greeting pools for morning, afternoon, evening, and night
- **Balance**: Low balance messages when wallet < ₦500, high balance messages when > ₦5000
- **General Rotation**: Falls back to general messages if no time-specific or balance-specific message is triggered

Messages rotate every **6 seconds** when displayed on the dashboard.
